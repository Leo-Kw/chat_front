# chat_front

### Quick start

```bash
git clone https://github.com/Leo-Kw/chat_front.git
# 安装依赖
cd chat_front && pnpm i
# 启动项目
pnpm dev
# 项目打包
pnpm build
```

### Note:

`form`组件是基于[react-hook-form](https://react-hook-form.com/get-started) 二次封装
